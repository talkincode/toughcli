# toughcli

toughcli 是一个基于toughradius以及相关软件的工具包。

## 运行环境

    - Linux/Mac OSX/BSD
    - Python 2.7/PyPy
    - Click

## 安装

    pip install toughcli

## 使用说明

    toughcli --help


    Usage: toughcli [OPTIONS] COMMAND [ARGS]...

    Options:
      --version
      --help     Show this message and exit.

    Commands:
      docker
      mysql
      radius
      redis
      upgrade
